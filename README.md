## What this is

My personal portfolio — designed in Figma, built in React. No template, no page builder. The site itself is part of the work sample: it demonstrates the same stack I use for client and product work.

I'm a UI Designer with 4+ years of B2B SaaS product experience who codes his own interfaces. This portfolio documents that overlap — from design system work in XAML to deployed React/TypeScript projects.

## Run locally

```bash
git clone https://github.com/Sebastian-Weber/Portfolio.git
cd Portfolio/frontend
npm install
npm run dev
```

Opens at `http://localhost:5173` (Vite default).
